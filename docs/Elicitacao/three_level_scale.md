# Three Level Scale

## Introdução
A priorização de requisitos é uma etapa essencial da engenharia de requisitos, pois orienta a definição das funcionalidades que devem ser implementadas primeiro. Nesse processo, as funcionalidades são organizadas de acordo com uma escala de três níveis, Baixa, Média e Alta, utilizando a técnica **Three Level Scale**.
